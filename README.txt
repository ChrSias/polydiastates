Η εργασία υλοποιεί και συγκρίνει τις δομές:
1. k-d Tree (μέσω scikit-learn)
2. Quad Tree (Custom Python implementation)
3. Range Tree (Custom Python implementation - 5 independent trees)
4. R-tree (μέσω libspatialindex/rtree)
5. LSH (Locality Sensitive Hashing) για αναζήτηση κειμένου

pandas
numpy
